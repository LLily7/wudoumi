| 革故鼎新       | 去掉旧的，建立新的                                           |

| 学富五车       | 学识渊博                                                     |
| 浩如烟海       | 形容文献资料书籍多得无法计算                                 |
| 汗牛充栋       | 藏书很多                                                     |
| 推陈出新       | 去掉旧事物的糟粕，取其精华，并使它向新的方向发展             |
| 驾轻就熟       | 对某事有经验，很熟悉，做起来容易                             |
| 审慎           | 考虑问题周密而谨慎                                           |
| 郑重           | 对待事物严肃认真                                             |
